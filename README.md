# weatherDashboard


Weather Dashboard built with React.js, Express.jS, and MySql

Features:

Fetching and displaying current temperature, weather conditions, wind speed, local time, and 12-hour weather forecast
Dynamic animations and background colors based on weather conditions / time of day
Auto refreshing of data, clean intuitive user interface
Dockerization
